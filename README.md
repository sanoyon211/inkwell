## 📖 Project Overview

**Inkwell** is a modern blogging platform designed for a seamless reading and writing experience. The platform features a clean typography-first design, allowing content to take center stage. Built with React for dynamic rendering and styled beautifully with Tailwind CSS.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js | Component-Based UI & Routing |
| JavaScript (ES6+) | Logic & Functionality |
| Tailwind CSS | Utility-First Styling |
| HTML5 | Markup Structure |
| CSS3 | Custom Design Elements |

---

## ✨ Key Features

- 📝 Clean blog post listing and detail view
- 🔠 Typography-first reading experience
- 📱 Fully **responsive** across all devices
- ⚡ Fast SPA with React Router navigation
- 🎨 Minimal and elegant UI design

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sanoyon211/inkwell.git

# 2. Navigate into the project folder
cd inkwell

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
# OR
npm start

# 5. Open in browser
# Visit: http://localhost:5173 or http://localhost:3000
```
